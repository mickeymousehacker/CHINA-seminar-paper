# Wiki — Ordnerstruktur

Jeder arbeitet **nur in seinem eigenen Ordner**. Claude liest alle Ordner.

| Ordner | Zuständig | Case |
|---|---|---|
| `byd/` | Batu | Vertikal integriertes Modell (BYD) |
| `jv-model/` | Kommilitone 1 | Joint Venture-Modell |
| `ma-model/` | Kommilitone 2 | M&A-getriebenes Modell |
| `shared/` | Alle | Gemeinsamer Vergleichsteil (nur via Pull Request) |

## Regeln
- Schreibe nur in deinem Ordner
- Für `shared/` immer einen Pull Request erstellen — kein direktes Pushen
- `git pull` vor jeder neuen Arbeitssitzung ausführen
